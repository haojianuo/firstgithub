1.if语句执行有个特点，它是从上往下判断，如果在某个判断上是True，把该判断对应的语句执行后，就忽略掉剩下的elif和else。例如：

	age = 20
	if age >= 6:
    		print('teenager')
	elif age >= 18:
    		print('adult')
	else:
    		print('kid')
以上结果为teenager
2.input()返回的数据类型是str，str不能直接和整数比较，必须先把str转换成整数。用int（）
	注意：int必须为整数，浮点数用float()
3.%转义数字。
		%d				整数
		%f				浮点数
		%s				字符串
		%x				十六进制整数
例如：
		print('%2d-%02d' % (3, 1))
		print('%.2f' % 3.1415926)
	结果：
		3-01
		3.14
